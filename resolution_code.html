Main Page of the website
Technicians entered data in the drop down menu based on the problem with the device
A Code was generated based on what they chose and put into the ticket
If there was an issue that was missing, you could "Request New Item" and it will direct you to a Microsoft Form. Once the form is filled out, it uploads to a central excel file and sends off an email saying an item needs to be reviewed

After the code was placed in a ticket, we used python and regular expressions to extract the data and create dashboards.
